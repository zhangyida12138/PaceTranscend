# PaceTranscend — 公式与枚举速查

供实现与单元测试对照，与 `docs/DESIGN.md` 一致。

## 灵力（未乘心境前）

| 来源 | 规则 |
|------|------|
| 活动能量 | 100 kcal → 100 灵力 |
| 锻炼时长 | 10 min → 50 灵力 |
| 步数 | 1000 步 → 20 灵力 |

## 心境倍率（示例合成顺序）

1. 睡眠少于 6h：×0.5
2. 睡眠 7–9h：×1.2（与「少于 6h」互斥时取明确规则优先级，建议以「昨夜主睡眠时长」单选一档）
3. 深度睡眠超过 20%：额外感悟（随机跳过小境界进度，与 EXP 倍率分开处理）
4. 当日 RHR 低于历史均值：灵力获取 ×1.1

## EXP 需求

\[
EXP_{level} = Base \times Major^3 \times (Minor \times 1.2)
\]

`Major` ∈ [1,9]，`Minor` ∈ {1,2,3,4} 对应 初/中/后/圆满。

## 惩罚数值

| 事件 | 效果 |
|------|------|
| 凌晨 2 点后仍未入睡 | −当前等级 2% EXP；次日灵力 ×0.5 |
| 2h 无站立提醒 | 扣少量灵力 +「经络淤堵」 |
| 连续 3 天无圆环闭合 | 大境界内从圆满降至后期 |

## 渡劫失败

−20% EXP，冷却 7 天后再触发渡劫流程。

## 大境界难度系数（可用于 Base 或奖励权重）

1.0, 1.5, 2.2, 3.5, 5.0, 8.0, 12.0, 20.0,（终局）
